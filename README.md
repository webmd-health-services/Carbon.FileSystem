The "Carbon.FileSystem" module...

# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'Carbon.FileSystem'
Import-Module -Name 'Carbon.FileSystem'
```

To install privately:

```powershell
Save-Module -Name 'Carbon.FileSystem' -Path '.'
Import-Module -Name '.\Carbon.FileSystem'
```
